Arquitetura AWS
  • Alta disponibilidade

    • Essa é uma arquitetura de alta disponibilidade e segurança baseada na AWS para hospedar uma aplicação web. Ela utiliza Global Accelerator e CloudFront para entregar conteúdo globalmente com baixa latência e proteção contra DDoS, além do Route 53 para gerenciar o DNS e redirecionar usuários para os endpoints corretos.

  • Segurança e Escalabilidade

    • A segurança é reforçada com IAM para controle de acessos, Security Groups para regras de firewall e proteção contra ataques como SQL Injection. A aplicação é hospedada no Amazon EC2 em uma sub-rede pública, enquanto os dados ficam no Amazon RDS em uma sub-rede privada, com backups automáticos para recuperação.

  • Monitoramento

    • CloudWatch e X-Ray garantem visibilidade do desempenho e identificam possíveis problemas.

  • Baixa Latência Global

    • Global Accelerator e CloudFront minimizam o tempo de resposta para usuários em diferentes regiões.
